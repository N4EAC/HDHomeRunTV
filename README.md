# HDHomeRun TV Player 1.0.8

Windows television-style player for SiliconDust HDHomeRun network tuners. No DRM.

## Playback engine

The program uses portable mpv and does not require VLC. On first playback it can install the engine under `%LOCALAPPDATA%\HDHomeRunTV\engine`.
